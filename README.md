## Run Locally

Clone the project

```bash
  git clone https://github.com/julubis/eduklimair
```

Go to the project directory

```bash
  cd eduklimair
```

Install dependencies

```bash
  npm install
```

Start the development server

```bash
  npm run start-dev
```
